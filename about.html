BIO

Hi, I'm Mike Hatch. A self-taught web developer based in mainland USA. I've focused mostly on front-end web technologies and dabbled a wee bit into the back-end.

Front-End:

HTML/CSS/JavaScript

The projects presented here demonstrate use of HTML 5, CSS (Flexbox/Grid/Animations/Bootstrap 4), and JavaScript (DOM/AJAX/JSON/jQuery). Skills not presently demonstrated but acquired are React and Express.

Back-End:

PHP

Although the front-end language JavaScript is my first, I've also spent some time with PHP. I was particularly interested in learning PHP to help me understand the inner workings of WordPress. Like everything else, it is a work in progress.

Content Management Systems:

WordPress

I really like WordPress and have used it on and off for many years as an end user. I'm able to navigate through the WordPress editor/admin area and make individual changes to the backend file structure. I plan to develop a site or two entirely in WP. I absolutely LOVE Gutenberg. Go blocks! Whoo!
